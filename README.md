# Certificate Portal

A complete system to apply, track, and download official certificates.

## Features
- Apply for various certificates with dynamic forms
- Receive an acknowledgement PDF with reference number
- Track application status online
- Admin panel with password protection (`08092003`)
- Review, edit, and approve/reject applications
- Generate and download official certificates

## Deployment
- Works on Render free tier with persistent disk
- All data stored in `data/data.json`

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Visit `http://localhost:3000`
